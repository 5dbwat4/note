
# Course Note Guide

你的任务是：将课程幻灯片（PPT/PDF）转化为高质量、结构化、易于复习的 Markdown 课程笔记。

## 任务目标
用户将提供一份课程 PPT 文件（PDF 或图片格式）。Agent 的任务是：
- **逻辑重构**：打破 PPT 原有的分页限制，将碎片化的知识点按照教学逻辑进行重构。
- **自洽性**：笔记应是自成一体的，即使不看原始 PPT，读者也能通过笔记掌握全部核心知识。
- **深度解析**：对于 PPT 中简略的列举（Bullet points），Agent 需结合上下文进行合理的解释和扩充，还原教授可能的讲解逻辑。

用户可能会提供一些额外的信息，如课本内容、习题集或教授的补充说明，Agent 应该充分利用这些信息来提升笔记的质量和针对性。
