# 幂级数


$e^{x}=$ <ReciteHidden> $\sum_{n = 0}^{\infty}\frac{x^{n}}{n!},x\in(-\infty,+\infty)$ </ReciteHidden>

$\sin x=$ <ReciteHidden> $\sum_{n = 1}^{\infty}\frac{(-1)^{n - 1}x^{2n - 1}}{(2n - 1)!},x\in(-\infty,+\infty)$ </ReciteHidden>

$\cos x=$ <ReciteHidden> $\sum_{n = 0}^{\infty}\frac{(-1)^{n}x^{2n}}{(2n)!},x\in(-\infty,+\infty)$ </ReciteHidden>

$\ln(1 + x)=$ <ReciteHidden> $\sum_{n = 1}^{\infty}\frac{(-1)^{n - 1}x^{n}}{n},x\in(-1,1]$ </ReciteHidden>

# 傅里叶级数

- $f(x)$ 以 $2T$ 为周期
- Fourier级数的表达式
   <ReciteHidden> $\frac{a_0}{2} + \sum_{n=1}^{+\infty} (a_n \cos \frac{n\pi x}{T} + b_n \sin \frac{n\pi x}{T})$ </ReciteHidden>
- Fourier级数的定义
  $(a_0, a_n, b_n, n \in \mathbb{Z}^+)$
  - $a_0 =$ <ReciteHidden> $\frac{1}{T} \int_{-T}^{T} f(x) dx$, </ReciteHidden>
  - $a_n =$ <ReciteHidden> $\frac{1}{T} \int_{-T}^{T} f(x) \cos \frac{n\pi x}{T} dx$, </ReciteHidden>
  - $b_n =$ <ReciteHidden> $\frac{1}{T} \int_{-T}^{T} f(x) \sin \frac{n\pi x}{T} dx$. </ReciteHidden>
