### 幂级数

#### 1. 幂级数的收敛区间
|名称|内容|
| ---- | ---- |
|幂级数|形如$\sum_{n = 0}^{\infty}a_{n}(x - x_{0})^{n}$的级数称为幂级数, 其中$x_{0}$是任意给定的实数, $a_{n}(n = 0,1,2,\cdots)$称为幂级数的系数|
|收敛半径|若幂级数$x = x_{0}\neq0$点收敛, 但不是在整个实轴上收敛, 则必存在一个正数$R$, 使得: ①当$\abs{x - x_{0}} < R$时, 幂级数收敛; ②当$\abs{x - x_{0}} > R$时, 幂级数发散. $(x_{0}-R,x_{0}+R)$称为幂级数的收敛区间|
|收敛半径的求法|设幂级数$\sum_{n = 0}^{\infty}a_{n}x^{n}$的收敛半径为$R$: 若$\lim\limits_{n\rightarrow\infty}\sqrt[n]{\abs{a_{n}}}=\rho$ (或$\lim\limits_{n\rightarrow\infty}\abs{\frac{a_{n + 1}}{a_{n}}}=\rho$或$\lim\limits_{n\rightarrow\infty}\sqrt[n]{\abs{a_{n}}}=\rho$):<br />(1) 当$0 < \rho<+\infty$时, 幂级数的收敛半径$R=\frac{1}{\rho}$;<br />(2) 当$\rho = 0$时, 幂级数的收敛半径$R = +\infty$;<br />(3) 当$\rho=+\infty$时, 幂级数的收敛半径$R = 0$|

#### 2. 幂级数的性质
|名称|内容|
| ---- | ---- |
|阿贝尔定理|若幂级数$\sum_{n = 0}^{\infty}a_{n}x^{n}$在$x = \overline{x}\neq0$处收敛, 则对满足不等式$\abs{x} < \abs{\overline{x}}$的任何$x$, 幂级数收敛而且绝对收敛; 若在$x=\overline{x}$发散, 则对满足不等式$\abs{x}>\abs{\overline{x}}$的任何$x$, 幂级数发散|
|和函数的连续性|若幂级数的收敛半径$R>0$, 则和函数$S(x)=\sum_{n = 0}^{\infty}a_{n}x^{n}$在收敛区间$(-R,R)$内连续|
|和函数的可积性与可导性|幂级数的和函数在其收敛域内的任一闭子区间上可积, 在其收敛区间内可导|
|一致收敛性|幂级数在其收敛域的任一闭子区间内一致收敛|
|逐项积分和导数|设幂级数在其收敛区间$(-R,R)$内的和函数为$f(x)$, 若$x$为$(-R,R)$内任意一点, 则:<br />(1) $f(x)$在$x$可导, 且$f^{\prime}(x)=\sum_{n = 0}^{\infty}na_{n}x^{n - 1}$;<br />(2) $f(x)$在$0$与$x$区间上可积, 且$\int_{0}^{x}f(t)dt=\sum_{n = 0}^{\infty}\frac{a_{n}}{n + 1}x^{n+1}$|
| |设①幂级数的和函数是$(-R,R)$内的连续函数; ②幂级数在收敛区间的左(右)端点上收敛, 则其和函数也在这一端点上右(左)连续|


### 3. 幂级数的运算

**定理**：若幂级数$\sum_{n = 0}^{\infty}a_{n}x^{n}$与$\sum_{n = 0}^{\infty}b_{n}x^{n}$的收敛半径为$R_{a}$和$R_{b}$, 则有
$$\lambda\sum_{n = 0}^{\infty}a_{n}x^{n}=\sum_{n = 0}^{\infty}\lambda a_{n}x^{n},|\ x|＜R_{a};$$
$$\sum_{n = 0}^{\infty}a_{n}x^{n}\pm\sum_{n = 0}^{\infty}b_{n}x^{n}=\sum_{n = 0}^{\infty}(a_{n}\pm b_{n})x^{n},|\ x|＜R;$$
$$\left(\sum_{n = 0}^{\infty}a_{n}x^{n}\right)\left(\sum_{n = 0}^{\infty}b_{n}x^{n}\right)=\sum_{n = 0}^{\infty}c_{n}x^{n},|\ x|＜R.$$
其中$\lambda$为常数,$R=\min\{R_{a},R_{b}\}$,$c_{n}=\sum_{k = 0}^{n}a_{k}b_{n - k}(n = 0,1,2,\cdots)$.

**定理**：若幂级数$\sum_{n = 0}^{\infty}a_{n}x^{n}$与$\sum_{n = 0}^{\infty}b_{n}x^{n}$在某邻域内相等, 则它们同次幂项的系数相等, 即$a_{n}=b_{n}(n = 0,1,2,\cdots)$.


## 函数的幂级数展开

## 1.泰勒级数

| 名称 | 内容 | 备注 |
|---|---|---|
| $ f(x) $ 在 $ x=x_{0} $ 处的泰勒级数 | 若函数 $ f(x) $ 在 $ x=x_{0} $ 处具有任意阶导数,则幂级数 <br />$ \sum_{n=1}^{\infty}\frac{f^{(n)}(x_{0})}{n!}(x-x_{0})^{n} $ 为函数 $ f(x) $ 在 $ x=x_{0} $ 处的泰勒级数<br />$ f(x)\sim\sum_{n=1}^{\infty}\frac{f^{(n)}(x_{0})}{n!}(x-x_{0})^{n} $ | 不能保证这个幂级数正 好收敛到 $ f(x) $ 本身 |
| $ f(x) $ 在区间上 可展开成它的 泰勒级数 | 设函数 $ f(x) $ 在点 $ x_{0} $ 具有任意阶导数,那么 $ f(x) $ 在区间 $ (x_{0}-r,x_{0}+r) $ 内关于它的泰勒级数的和函数的充分条件是:对一 切满足不等式 $ \mid x-x_{0}\mid<r $ 的 $ x $ 有 $ \lim_{n\rightarrow\infty}R_{n}(x)=0 $. $ R_{n}(x) $ 为泰勒余项 |  |
| 函数泰勒级数 展开的充分条 件 | 设函数 $ f(x) $ 在 $ x_{0} $ 处具有任意阶导数, $ f(x) $ 在区间 $ (x_{0}-r,x_{0}+r) $ 内的其和函数的充分条件是 $ \lim_{n\rightarrow\infty}R_{n}(x)=0 $,其中 $ R_{n}(x) $ 是 $ f(x) $ 在 $ x_{0} $ 处的泰勒公式中的余项<br />$ \forall x\in(x_{0}-r,x_{0}+r) $ | 余项对函数能否展开为 幂级数的重要性 |
| 麦克劳林级数 | $ x_{0}=0 $ 处的泰勒展开式:<br />$ f(0)+f^{\prime}(0)x+\frac{f^{\prime\prime}(0)}{2!}x^{2}+\cdots+\frac{f^{(n)}(0)}{n!}x^{n}+\cdots $ | 麦克劳林公式在工程应 用中十分重要 |

### 2. 初等函数的幂级数展开式

$e^{x}=\sum_{n = 0}^{\infty}\frac{x^{n}}{n!},x\in(-\infty,+\infty)$

$\sin x=\sum_{n = 1}^{\infty}\frac{(-1)^{n - 1}x^{2n - 1}}{(2n - 1)!},x\in(-\infty,+\infty)$

$\cos x=\sum_{n = 0}^{\infty}\frac{(-1)^{n}x^{2n}}{(2n)!},x\in(-\infty,+\infty)$

$\ln(1 + x)=\sum_{n = 1}^{\infty}\frac{(-1)^{n - 1}x^{n}}{n},x\in(-1,1]$

$(1 + x)^{a}=1+\sum_{n = 1}^{\infty}\frac{a(a - 1)\cdots(a - n + 1)}{n!}x^{n},x\in(-1,1),a\in\mathbf{R}$

$\frac{1}{1 - x}=\sum_{n = 0}^{\infty}x^{n},\vert x\vert<1$

$\frac{1}{1 + x}=\sum_{n = 0}^{\infty}(-1)^{n}x^{n},\vert x\vert<1$
