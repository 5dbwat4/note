# 课程记录：72h货通全球 矢志创新创业

**主讲嘉宾：** 万霖 （菜鸟集团CEO、阿里巴巴合伙人）

### 一、 开场与个人经历分享

#### 1.1 引言
万霖老师首先表达了对再次来到浙大课堂的荣幸与喜悦。这是他第三次参与该课程，面对在座的同学们，他回忆起自己的大学时光，认为大学阶段是充满可能性、不断尝试与思考未来的时期。

#### 1.2 教育背景与早期探索
- **学习经历**：万霖老师是武汉人，本科和研究生就读于武汉，主修**造船专业**。后赴美留学，在**德克萨斯大学奥斯汀分校**获得博士学位，主攻**运筹学**，研究方向为优化算法。
- **大学时期的探索**：他坦言自己并非一开始就目标明确的人，高考和选专业时也并非十分清晰。在大学期间，他进行了广泛的尝试，如考取高级程序员、辅修经济法双学位、参加律师资格考试等，鼓励同学们在大学阶段多做尝试。
- **研究生时期的关键课题**：他分享了一个已解密的硕士课题——参与将“东风快递”（战略导弹）移植到潜艇上的项目，负责设计一个在冷发射过程中保护火箭喷射口的“铝合金造纸”（此处应为口误，实为保护罩）。正是这个项目，让他接触到了优化算法，从而引导他走向了运筹学的研究道路。

#### 1.3 大学教育的核心价值
万霖老师认为，大学里学到的具体知识在工作中未必能直接应用，但有两样东西至关重要，是真正的“大学之道”：
1.  **解决问题的能力**：
    -   他指出，在任何地方，**提出问题的人永远比解决问题的人多**。
    -   真正具有解决复杂问题能力的人是稀缺的。他建议同学们不管学习什么专业，都要注重培养将复杂问题化繁为简并逐步解决的能力，这是终生受用的本领。
2.  **思考问题的方式（运筹学思维）**：
    -   这正是他从运筹学中受益最深的思维框架：**在有限的环境和资源约束下，清晰定义目标函数，并寻找最优解**。
    -   无论是最大化利润、提升效率，还是其他目标，所有事情都可以套用这个模型，它能帮助自己明确努力方向。

#### 1.4 职业生涯：从亚马逊到菜鸟
- **亚马逊时期（九年）**：
    -   他于2005年加入亚马逊，恰逢公司经历互联网泡沫后进入第二个高速增长期，每年保持约30%的增速。
    -   **平台的力量**：他深刻感受到一个快速成长的平台会倒逼和推动个人成长。很多问题是全新的，所有人都没有答案，大家被倒逼着去建立认知、解决问题。
    -   **打破刻板印象与自我突破**：在面临技术和管理岗位的双重机会时，他主动跳出舒适圈，从一名科学家（Scientist）逐步走向更综合的管理岗位。
    -   **工作心法**：他总结了两点，一是务必在动手前真正**理解工作的实质和期望值**；二是交付成果时，**永远比期望值好一点点**，以此建立个人“靠谱”的品牌。
- **加入菜鸟的决策点（2014年）**：
    -   **兴奋的体量**：当时亚马逊全球日包裹量约400万个，而中国日常包裹已达2000万个，双11当天更是高达1亿个。物流的网络效应和规模效应让他看到了巨大的机会和挑战。
    -   **人生的折腾**：39岁时，他决定做出改变。怀抱“人生需要折腾一下”的信念，抱着好奇与兴奋感，仅用几个月便卖掉房车，举家搬回国内。
    -   **加入的契机**：作为当时亚马逊职位最高的华人之一，经由猎头和同事的推荐，他与阿里进行了接洽，并被菜鸟的故事深深吸引。

### 二、 菜鸟的创业历程：基于客户价值的创新

#### 2.1 菜鸟的诞生背景与愿景
-   **诞生驱动**：2012年双11商业上取得巨大成功，但物流端成为“噩梦”。当时行业处于手工化阶段，无法应对电商的爆发式增长。菜鸟于2013年应运而生，是为解决电商快速发展与落后物流基础设施之间的矛盾。
-   **三大关键词**：
    1.  **数字化**：将互联网技术、数据技术与物流产业深度融合，这是解决爆发式电商增长的根本路径。
    2.  **全球化**：中国企业全球化需要与之匹配的全球物流网络。当时全球成规模的物流网络（UPS，FedEx，DHL）均伴随美国企业全球化而崛起，菜鸟致力于建设一张属于中国的全球化物流网络。
    3.  **产业化**：物流最终必须落实到线下的实体交付，因此必须是一个运营驱动的产业化网络。
-   **愿景（PPT第5页画外音）**：我们的愿景是成为全球领先的物流网络解决方案提供商，为全球客户提供高效、可靠、可持续的物流解决方案。
-   **长期目标**：构建一张技术驱动的、能匹配电商发展的、普惠的**全球72小时必达**网络。他强调，关键不仅在于“72小时”这个时效，更在于“普惠”和“匹配电商发展”，这意味着成本必须从传统快递的50美金降至电商可承受的5美金级别。

#### 2.2 国内创新案例：基于痛点打造产品
万霖老师指出，创新的起点是**基于客户价值**，而非自嗨。创新不仅要解决问题，还要**创造出足够大的价值**，如此才能构建可持续的商业模式去分配价值。

**案例一：电子面单——包裹的数字化**
-   **背景与痛点**：2013年前后，中国快递虽体量庞大（日均2000万-2500万包裹），但操作模式原始，依赖“大头笔”和手写五联面单。最关键的角色是能用0.1秒识别手写地址并标注编码的分拣员（“人工智能”）。这种模式无法在双11等大促期间弹性扩容，即使快递公司有资金购买先进的交叉带分拣机，也因系统无法识别面单而无法使用。
-   **解决方案**：菜鸟推出的**电子面单**项目，通过算法（当时还未称AI）将地址数字化，生成了一个包含“**三段码**”（分拨中心、站点、快递员）的统一标准格式。包裹在生成面单的那一刻，其全链路路由就已确定。
-   **价值与影响**：电子面单使每个包裹都有了系统可识别的“身份证”，为后续大规模上线自动化分拣设备铺平了道路。2-3年内，手写面单基本被替换，中国也成为全球交叉带分拣机保有量最高的国家，许多分拨中心实现了“黑灯工厂”的端到端自动化。

**案例二：菜鸟裹裹——快递员的数字化**
-   **背景与痛点**：线上消费增长带来退货需求激增，但逆向物流成本高、体验差。没有快递员愿意做退货（单点揽收效率低、成本高），消费者面临取货时间不确定、价格贵且不透明、退款周期长等问题（PPT第9页）。
-   **解决方案**：推出“菜鸟裹裹”服务，成为**中国最大的众包寄件平台**（PPT第9页）。通过数字化调度，将分散的退货需求进行实时路由规划，实现了高效、准时、低成本的揽收，并向消费者承诺“**最快30分钟上门揽件**”，联合淘宝天猫推出“**极速退款服务**”。
-   **价值与影响**：通过模式创新和成本优化，将原本无人愿做、不赚钱的退货生意，变成了一个可盈利、可规模化、多方共赢的业务。

### 三、 全球化：长期主义的胜利

#### 3.1 全球化面临的挑战与心态
-   **挑战**：全球有200多个国家，每个国家在语言、文化、法规（尤其是劳工保护）上都存在巨大差异，如巴西对劳工的保护甚至超越发达国家。
-   **心态：“陆止于此，海启于斯”**：他分享了在葡萄牙罗卡角的见闻，那块碑既是欧亚大陆的尽头，也是大航海时代的起点。这启示我们，面对全球化这一充满挑战的“大海”，应将其看作一个充满机遇的新起点，而非不可逾越的边界。

#### 3.2 跨境物流的机遇与痛点
-   **市场趋势（PPT第12页）**：全球跨境电商市场规模高速增长，物流流向从单一的“中国到全球”变得更加丰富，发货模式也趋于多元（直发、海外仓、本地发货）。
-   **核心痛点（PPT第13页）**：早期跨境物流产品存在“慢、贵、不可靠”的痛点。例如，发往巴西的包裹曾需要**45天**甚至更久，且追踪信息缺失。要么选择昂贵的三大国际快递（50美金以上），要么忍受慢且不靠谱的万国邮联。
-   **用户启示**：在巴西世界杯期间，尽管卖家公告包裹可能**延迟至90天**，但消费需求依然旺盛，这证明了跨境电商的刚性需求和巨大潜力。

#### 3.3 长期主义的产品落地与能力建设
-   **破局产品：“5美元10日达”**：万霖老师团队从客户角度出发，设定了“**一杯咖啡的价格（5美元）**，实现**确定性的10日送达并配备端到端追踪**”这一目标。该产品精准匹配了当时跨境电商20美金左右的客单价和能承受的物流成本，一经推出便大获成功，第一年日均包裹量即突破100万个。
-   **建设全球智慧物流骨干网（PPT第16页）**：支撑“5美元10日达”的是底层长期、扎实的能力建设，主要包括：
    1.  **全球首公里**：建成中国产能最大的揽收分拨网络。
    2.  **航空干线**：每周运营30多架次包机，开通50多条固定运力航线。
    3.  **eHub枢纽（以比利时列日为例）**：
        -   **选址逻辑**：综合考虑了地理覆盖（陆运次日达覆盖欧洲70%人口）、机场属性（以货运为主，无宵禁）等因素。
        -   **eHub的集成功能（PPT第17页）**：在一个场地内集成提货、拆板、清关、分拨（**高效集成**），通过AI赋能的**智慧清关**实现数据直连、合规高效，并通过多式联运实现**一点联通全球**。
        -   **逆行者的故事（PPT第18页）**：新冠疫情期间，在所有人从欧洲撤回中国时，菜鸟同学逆行前往比利时，确保项目按时交付。如今，列日eHub不仅成为菜鸟欧洲最大的智慧物流枢纽，也带动列日机场跃升为欧洲货运量前五的机场，展现了物流的网络集聚效应。
    4.  **全球关务**：覆盖60多个国家和地区，对接70多个口岸，实现全球预报、智能审单、快速通关。
-   **海外本地网络建设（PPT第19-21页）**：
    -   **自建网络的动因**：在与西班牙邮政等当地伙伴合作时，因增长逻辑错位（我方希望量大价优，对方因加班成本要求提价）而决心进行自建。
    -   **全球落地**：目前已在**10个国家和地区**（如西班牙、法国、墨西哥、以色列、巴西等）建立起本地快递能力，并复制“菜鸟驿站”和自助柜等成熟模式。例如，在以色列，消费者对自助式驿站的高接受度验证了这一模式的全球可复制性。
    -   **海外仓网**：在全球布局了近**100万平方米**的海外仓，处于行业第一梯队，以满足品牌商家将库存前置到离消费者最近地方的供应链需求。

### 四、 物流科技：技术与场景深度融合

#### 4.1 科技成为核心竞争力（PPT第22页）
科技不但是菜鸟优化自身运营效率的支柱，更成为对外输出的核心差异化竞争力。通过将内部能力产品化，为外部客户提供经济高效的技术解决方案。

#### 4.2 科技输出案例
-   **助力宝洁全球仓储自动化升级（PPT第23页）**：菜鸟已成为宝洁全球自动化的战略合作伙伴，将其自主研发的世界级自动化方案应用于宝洁全球50多个仓库的升级改造中。
-   **与麦当劳合作的RFID项目（PPT第24页）**：针对中国食品监管“一物一码”的要求和麦当劳供应链各环节信息不通、盘点低效的痛点，菜鸟部署了端到端的RFID物联网解决方案，实现了全链路信息可追溯、餐厅收货盘点效率提升30%，成为餐饮行业首个大规模RFID应用的案例。
-   **前沿技术探索**：研发新一代“攀爬机器人”，目标是合并货架存储和地面搬运两套系统，实现统一调度，使整体效率再提升20%-30%。万霖老师认为，物流行业因其规模大、标准化程度高的特点，会成为AI和机器人技术率先应用并取得突破的理想场景。

### 五、 社会责任：成为一家好公司

万霖老师认为，一家公司除了商业价值，更重要的是**成为一家好公司**，让一群有情有义的人共同做一件有价值、有意义的事。菜鸟聚焦于**五大责任领域（PPT第27页）**，并将其融入战略和运营。
1.  **绿色物流**：
    -   **定位**：以绿色供应链和绿色回收为特色，做循环经济的领头羊。
    -   **成果（PPT第28页）**：菜鸟自身运营和价值链共计减排**45.8万吨**。具体措施包括：自营城市配送新能源车占比达**99%**、安装光伏**47.9兆瓦**、清洁电力使用比例达**54.1%**、简约包装应用于**1.3亿个**包裹、原箱发货超**4.3亿个**、重复再使用旧纸箱**4755.8万个**等。
2.  **应急物流**：
    -   **定位**：成为应急物流领域的专家。
    -   **实践**：新冠疫情初期，菜鸟第一时间开通免费救援物资运输专线，初期将全球物资运往中国。武汉疫情爆发后，又反向将物资运送至全球。这背后是强大的数字化能力和全球网络的支撑。菜鸟也成为了**中国首家联合国世界粮食计划署全球物流合作伙伴**。
3.  **用户体验**：显著提升消费者满意度，成为值得信赖的物流服务提供商。
4.  **高质量就业**：打造有产业互联网特色的就业模式，让员工有幸福感。他特别提到，年轻员工的第一诉求是**成长**，因此公司必须提供学习、轮岗和参与全球化的机会。
5.  **服务社区**：
    -   **定位**：立足业务属地，做社区好公民。
    -   **实践（PPT第29页）**：在全国组建22支“鸿雁”志愿者队伍。在比利时，通过赞助当地足球队等方式积极融入社区；列日eHub为当地创造了约**300个**就业机会。

### 六、 菜鸟文化、总结与寄语

#### 6.1 菜鸟的人才特质（PPT第30页）
万霖老师亲自定义了菜鸟人所需的“四劲”：
1.  **使命驱动的韧劲**：因所做事业需长期投入，必须对目标高度认同，百折不挠。
2.  **迎难而上的狠劲**：物流难题无法绕行，如应对双11的天量包裹，必须直面挑战并解决。
3.  **精益求精的干劲**：在极度内卷的环境下，只做到六七十分的产品没有意义，必须做到极致。
4.  **以客户价值出发的巧劲**：创新成本最高，但必须做。创新的唯一出发点是**从客户价值出发**，解决真痛点，创造真价值，以此提高创新的成功率。

#### 6.2 课程总结：创新创业的四大关键（PPT第31页）
1.  **从客户价值出发，打造极致产品**，真正解决客户痛点，创造足够价值。
2.  **坚持长期主义，做难而正确的事情**，从底层能力建设形成护城河，并认识到护城河只是局部的时间差，需不断迭代。
3.  **技术创新必须和业务场景深度融合**，才能发挥真正价值。他提醒我们，人们往往高估技术的短期影响，而低估其长期（5-10年）带来的颠覆性变化。
4.  **成为一家好公司**，做有价值有意义的事情，这是凝聚团队的关键。

#### 6.3 寄语（PPT第32页）
最后，万霖老师以菜鸟的slogan寄语在座同学：
**“秉持菜鸟之心，超越鸿鹄之志”**
始终保持谦逊的空杯心态，同时志存高远，去解决行业的真问题，去建设全球化的物流网络。

### 七、 互动问答精要

1.  **关于运筹与供应链优化的未来方向**：数字化仍未解决所有痛点，供应链高度定制化且复杂。随着AI，特别是智能体（Agent）的发展，有望极大降低定制化成本，重塑供应链能力，这其中潜力巨大。
2.  **关于亚马逊与阿里的文化差异**：亚马逊是强管控、强自营的TOP-DOWN文化，类似京东；而阿里则是开放平台、生态化、更草根和自下而上（BOTTOM-UP）的文化。两者在商业模式和管理文化上差异显著。
3.  **关于AI时代年轻人应强化的能力**：与其焦虑被AI取代，不如担忧被“更会用AI的人”取代。应做“早期使用者”，比他人早几个月建立对AI的深度认知，这本身就是巨大优势。
4.  **关于无人配送和机器人落地的判断**：硬件（运动控制等）的进展快于软件（对物理世界的认知）。瓶颈在于缺乏基于“世界模型”的物理世界大模型，且无海量数据训练。物流因其场景相对受限、不载人，是机器人技术可以率先取得突破的领域之一。
