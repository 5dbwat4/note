  <Q>
    当用一个对象去初始化同类的另一个对象时,要调用拷贝构造函数。
  </Q>
  <Explanation>**正确**。拷贝构造函数的定义即是：用一个已存在的同类对象去初始化一个新对象时调用的构造函数。例如 `AA d(c);` 或 `AA e = d;` 都是初始化而非赋值，会触发拷贝构造函数。赋值（如 `a1 = a2;`）则调用赋值运算符而非拷贝构造函数。</Explanation>

<PTAQuiz true-or-false title="对象间赋值将调用拷贝构造函数。  " correct={false} meta={{label: "1-2", score: 2, difficulty: 0, author: "黄淑伟", org: "沈阳师范大学"}}>
  <Q>
    对象间赋值将调用拷贝构造函数。
  </Q>
  <Explanation>**错误**。对象间赋值调用的是**赋值运算符**（`operator=`），而非拷贝构造函数。拷贝构造函数仅在**初始化**新对象时调用（如 `AA b(a);` 或 `AA b = a;`），而赋值发生在对已构造对象进行操作时（如 `a = b;`），两者触发时机不同。</Explanation>
</PTAQuiz>

<PTAQuiz multiple-choice title="假设MyClass是一个类，则该类的拷贝初始化构造函数的声明语句为（ ）    " correct={2} meta={{label: "2-3", score: 2, difficulty: 0, author: "范鹏程", org: "内蒙古师范大学"}}>
  <Q>
    假设MyClass是一个类，则该类的拷贝初始化构造函数的声明语句为（ ）

  </Q>
  <C>`MyClass&(MyClass x);`</C>
  <C>`MyClass(MyClass x);`</C>
  <C>`MyClass(MyClass &x);`</C>
  <C>`MyClass(MyClass *x);`</C>
  <Explanation>
  答案为 **`MyClass(MyClass &x);`**。拷贝构造函数的标准签名形式为 `ClassName(const ClassName &obj);`。  

- **A** `MyClass&(MyClass x);` → 返回引用类型，不是构造函数。  
- **B** `MyClass(MyClass x);` → 参数按值传递，调用时会无限递归（传值本身就需要拷贝构造），编译器禁止这种写法。  
- **C** `MyClass(MyClass &x);` → 正确，参数为同类型对象的引用（最好加上 `const`）。  
- **D** `MyClass(MyClass *x);` → 参数为指针，不是拷贝构造函数签名。
</Explanation>
</PTAQuiz>

<PTAQuiz multiple-choice title="假设A是一个类的名字,下面哪段程序不会用到A的拷贝构造函数？   " correct={0} meta={{label: "2-6", score: 2, difficulty: 0, author: "黄淑伟", org: "沈阳师范大学"}}>
  <Q>
    {`
    假设A是一个类的名字,下面哪段程序不会用到A的拷贝构造函数？

    B. void func( A a) { cout<<"good"<< endl; }
    C. A func() { A tmp; return tmp;}

    `}
  </Q>
  <C>`A a1,a2; a1=a2;`</C>
  <C>`void func( A a) { cout<<"good"<< endl; }`</C>
  <C>`A func() { A tmp; return tmp;}`</C>
  <C>`A a1; A a2(a1);`</C>
  <Explanation>
  答案为 **A（`A a1,a2; a1=a2;`）**，该段程序不会用到拷贝构造函数。  

- **A** `A a1,a2;` → 两个对象均通过默认构造函数创建；`a1=a2;` → **已存在对象间的赋值**，调用的是赋值运算符而非拷贝构造函数。✓ 不用拷贝构造。  
- **B** `void func(A a) { ... }` → 按值传递对象参数，形参通过拷贝构造函数从实参初始化。  
- **C** `A func() { A tmp; return tmp; }` → 按值返回对象，临时对象通过拷贝构造函数从局部对象 `tmp` 初始化。  
- **D** `A a1; A a2(a1);` → 显式用 `a1` 拷贝构造 `a2`，调用拷贝构造函数。  

记忆要点：带 `=` 的初始化和赋值区分——等号在**定义语句**中是初始化（拷贝构造），在**已定义对象之后**是赋值。
</Explanation>
