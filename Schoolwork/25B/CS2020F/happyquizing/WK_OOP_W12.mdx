  <Q>
    `pair`类模板的作用是将两个数据组成一个数据，用来表示一个二元组或一个元素对，两个数据可以是同一个类型也可以是不同的类型。
  </Q>
  <Explanation>
    正确。`pair` 类模板定义在 `<utility>` 头文件中，它将两个值（`first` 和 `second`）组合成一个单元。第一个元素和第二个元素的类型可以是相同的（如 `pair<int,int>`），也可以是不同的（如 `pair<int,string>`）。`pair` 常用于需要返回两个值的场景，比如 `map` 的键值对。
  </Explanation>

<PTAQuiz multiple-choice title="Given:" correct={3} meta={{label: "2-2", score: 2, difficulty: 0, author: "翁恺", org: "浙江大学"}}>
  <Q>
    Given:
    ```
    template < class T >
    void swap( T& x, T& y ) {
       T temp = x;
       x = y;
       y = temp;
    }
    int i,j;
    float f,m;
    ```
    Which statement is incorrect?

  </Q>
  <C>`swap(i,j);`</C>
  <C>`swap(j,i);`</C>
  <C>`swap(f,m)`</C>
  <C>`swap(i,f);`</C>
  <Explanation>
    正确答案：`swap(i,f);`。函数模板 `swap` 只有一个模板参数 `T`，在调用时编译器根据实参类型推导 `T`。`swap(i,f)` 中 `i` 为 `int`，`f` 为 `float`，导致 `T` 无法同时推导为两种不同类型，产生类型冲突。`swap((int)c,i)`（若存在）或显式指定 `swap<int>(i,f)` 无效，因为 `T&` 不能绑定不同类型的引用。选项 A、B、C 中两个实参类型一致，均可正确推导和调用。
  </Explanation>
</PTAQuiz>
