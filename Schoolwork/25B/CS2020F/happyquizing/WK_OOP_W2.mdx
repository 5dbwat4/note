  <Q>
    若有下面的语句：
    ```cpp
    vector<int> v;

        v.push_back(i + 1);
    cout << v.size() << endl;
    ```
    则执行后程序的输出结果是
  </Q>

  <C>2</C>

  </Explanation>

<PTAQuiz multiple-choice title="vector构造函数调用时机" correct={2} meta={{label: "2-2", score: 2, difficulty: 0, author: "孔德桢", org: "浙大城市学院"}}>
  <Q>
    设有定义 `vector<string> v(10);`
    执行下列哪条语句时会调用构造函数？
  </Q>
  <C>`v[0] += "abc";`</C>
  <C>`v[0] = "2018";`</C>
  <C>`v.push_back("ZUCC");`</C>
  <C>`cout << (v[1] == "def");`</C>
  <Explanation>
    **答案：`v.push_back("ZUCC");`**。`vector<string> v(10)` 已创建了10个空字符串对象。`v[0] += "abc"` 和 `v[0] = "2018"` 均修改已存在的元素，不调用 `string` 的构造函数；`cout << (v[1] == "def")` 仅进行比较操作。而 `push_back("ZUCC")` 需要将字符串字面量 `"ZUCC"` 转换为 `string` 临时对象再插入，因此会调用 `string` 的构造函数。A、B、D均不涉及新对象的构造。
  </Explanation>
</PTAQuiz>

<PTAQuiz multiple-choice title="map与char*键" correct={1} meta={{label: "2-3", score: 2, difficulty: 0, author: "孔德桢", org: "浙大城市学院"}}>
  <Q>
    设有如下代码段：
    ```cpp
    std::map<char *, int> m;
    const int MAX_SIZE = 100;
    int main() {
        char str[MAX_SIZE];
        for (int i = 0; i < 10; i++) {
            std::cin >> str;
            m[str] = i;
        }
        std::cout << m.size() << std::endl;
    }
    ```
    读入10个字符串，则输出的 `m.size()` 为
  </Q>
  <C>0</C>
  <C>1</C>
  <C>10</C>
  <Explanation>
    **答案：1**。`std::map<char*, int>` 使用 `char*` 作为键，而 `char*` 的默认比较是比较**指针地址**而非字符串内容。代码中10次输入都读入同一个字符数组 `str`（其地址始终不变），因此 `m[str] = i` 每次操作的键都是同一地址，后续赋值会覆盖前一次的值。最终 map 中只有1个键值对，`m.size()` 为1。若想按字符串内容索引，应使用 `std::map<std::string, int>`。
  </Explanation>
</PTAQuiz>

<PTAQuiz multiple-choice title="创建vector的错误方法" correct={3} meta={{label: "2-5", score: 2, difficulty: 0, author: "刘莹", org: "威海职业学院"}}>
  <Q>
    下列创建 `vector` 容器对象的方法中，错误的是。

    A. `vector<int> v(10);`
    B. `vector<int> v(10, 1);`
    C. `vector<int> v{10, 1};`
    D. `vector<int> v = (10, 1);`
  </Q>
  <C>`vector<int> v(10);`</C>
  <C>`vector<int> v(10, 1);`</C>
  <C>`vector<int> v{10, 1};`</C>
  <C>`vector<int> v = (10, 1);`</C>
  <Explanation>
    **答案：D（`vector<int> v = (10, 1);`）**。`(10, 1)` 是C++的逗号运算符，先计算 `10`（丢弃），再计算 `1`，整个表达式的值为 `1`。因此该语句等价于 `vector<int> v = 1;`，而 `vector<int>` 没有接受单个 `int` 的转换构造函数，导致编译错误。A正确：`vector<int> v(10)` 创建10个默认值0的元素；B正确：`vector<int> v(10, 1)` 创建10个值均为1的元素；C正确：`vector<int> v{10, 1}` 使用列表初始化，创建包含元素 `{10, 1}` 的`vector`。
  </Explanation>
</PTAQuiz>

<PTAQuiz multiple-choice title="哪一项不是迭代器" correct={3} meta={{label: "2-6", score: 2, difficulty: 0, author: "刘莹", org: "威海职业学院"}}>

  </Explanation>
