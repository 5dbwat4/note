<PTAQuiz true-or-false title="In C++, inheritance allows a derived class to directly access al" correct={false} meta={{label: "1-1", score: 1, difficulty: 0, author: "翁恺", org: "浙江大学"}}>
  <Q>
    In C++, inheritance allows a derived class to directly access all of the functions and data of its base class.
  </Q>
  <Explanation>
    该说法**错误**。

    派生类通过继承**不能**直接访问基类的 `private` 私有成员。基类的私有成员对派生类是不可见的，只有基类的 `public` 和 `protected` 成员才能被派生类直接访问。

    总结：继承的"可见性"受基类访问控制符限制——`private` 成员不会被子类继承访问，`public` 和 `protected` 成员可以。

    示例：
    ```cpp
    class Base {
    private:
        int x;   // 派生类无法直接访问
    protected:
        int y;   // 派生类可以直接访问
    public:
        int z;   // 派生类可以直接访问
    };
    ```
  </Explanation>
</PTAQuiz>

<PTAQuiz true-or-false title="One class can have more than one super classes.  " correct={true} meta={{label: "1-2", score: 1, difficulty: 0, author: "翁恺", org: "浙江大学"}}>
  <Q>
    One class can have more than one super classes.
  </Q>
  <Explanation>
    该说法**正确**。

    C++ 支持**多继承**（multiple inheritance），一个派生类可以同时继承自多个基类（即拥有多个"父类"/"super class"）。

    语法示例：
    ```cpp
    class Derived : public Base1, public Base2, public Base3 {
        // 同时继承 Base1、Base2、Base3
    };
    ```

    需要注意：多继承可能引入**二义性**（如菱形继承问题），C++ 通过虚基类（`virtual` inheritance）来解决此类问题。但本题仅讨论"一个类是否可以有多个父类"——答案是肯定的。
  </Explanation>
</PTAQuiz>
