## 一致收敛性

### 1. 有关函数列的定义

| 名称   | 定义 |
| ------ | ---- |
| 函数列 | 设 $f_1(x), f_2(x), \cdots, f_n(x), \cdots$ 是一列定义在同一数集 $E$ 上的函数，则称其为定义在 $E$ 上的函数列，记作 $\{f_n\}$ 或 $f_n (n=1,2,\cdots)$ |
| 收敛点 | 设 $x_0 \in E$，将 $x_0$ 代入函数列得到 $f_1(x_0), f_2(x_0), \cdots, f_n(x_0), \cdots$ 若此数列收敛，则称函数列在点 $x_0$ 收敛，称 $x_0$ 为函数列的收敛点；若此数列发散，则称函数列在点 $x_0$ 发散 |
| 收敛域 | 使函数列 $\{f_n(x)\}$ 收敛的全体收敛点的集合，称为函数列 $\{f_n(x)\}$ 的收敛域 |
| 一致收敛性 | 设函数列 $\{f_n(x)\}$ 与函数 $f(x)$ 定义在同一数集 $D$ 上，若对任意的正数 $\epsilon$，存在某正整数 $N$，使得当 $n > N$ 时，对一切 $x \in D$，有 $\abs{f_n(x) - f(x)} < \epsilon$，则称函数列 $\{f_n(x)\}$ 在 $D$ 上一致收敛于 $f(x)$，记作 $f_n(x) \xrightarrow{\text{一致}} f(x) (n \to \infty), x \in D$ |

> 小提示，在考虑函数列的时候，可以思考一下数列与函数列、数项级数与函数项级数这四者之间的联系。

### 2. 有关函数项级数的定义

| 名称   | 定义 |
| ------ | ---- |
| 函数项级数 | 级数 $\sum_{n=1}^{\infty} u_n(x) = u_1(x) + u_2(x) + \cdots + u_n(x) + \cdots$，其中每一项 $u_1(x), u_2(x), \cdots, u_n(x), \cdots$ 都是定义在 $E$ 上的一个函数，则称级数为函数级数 |
| 和函数 | 若 $\forall x \in D \subseteq E, S(x) = \lim_{n \to \infty} S_n(x) = \lim_{n \to \infty} \sum_{k=1}^{n} u_k(x)$ 存在，则称级数 $\sum_{n=1}^{\infty} u_n(x)$ 在 $D$ 上收敛，其和函数为 $S(x)$ |
| $\sum u_n(x)$ 在 $D$ 上一致收  | 若 $\sum_{n=1}^{\infty} u_n(x)$ 的部分和函数列 $S_n(x) = \sum_{k=1}^{n} u_k(x)$ 在 $D$ 上一致收敛于 $S(x)$，则称 $\sum_{n=1}^{\infty} u_n(x)$ 在 $D$ 上一致收敛于 $S(x)$ |


### 3. 一致收敛的准则

| 名称   | 内容 |
| ------ | ---- |
| 柯西准则 | 函数列 $\{f_n(x)\}$ 在数集 $D$ 上一致收敛的充要条件是：对任给正整数 $\epsilon$，总存在正整数 $N$，使得当 $n, m > N$ 时，对一切 $x \in D$ 都有 $\abs{f_n(x) - f_m(x)} < \epsilon$ |
| 最值判别法 | $f_n(x) \xrightarrow{D} f(x)$ 的充要条件是：$\lim_{n \to \infty} \sup_{x \in D} \abs{f_n(x) - f(x)} = 0$ |
| 柯西准则 | 函数项级数 $\sum u_n(x)$ 在 $D$ 上一致收敛的充要条件是：$\forall \epsilon > 0$，存在与 $x$ 无关的 $N$，当 $n > N$ 时，对 $\forall x \in D$ 和一切正整数 $\theta$ 都有 $\abs{S_{n+\theta}(x) - S_n(x)} < \epsilon$ 或 $\abs{\sum_{k=n+1}^{n+\theta} u_k(x)} < \epsilon$ |
| 最值判别法 | $\sum S_n(x) \xrightarrow{D} S(x)$ 的充要条件是：$\lim_{n \to \infty} \sup_{x \in D} \abs{S(x) - S_n(x)} = 0$ |
| M判别法（魏尔斯特拉斯判别法） | 设函数项级数 $\sum u_n(x)$ 定义在数集 $D$ 上，$\sum M_n$ 为收敛的正项级数，若对一切 $x \in D$ 和 $n \in \mathbb{N}_+$ 有 $\abs{u_n(x)} \leq M_n, (n=1,2,\cdots)$，则函数项级数 $\sum u_n(x)$ 在 $D$ 上一致收敛 |
| 阿贝尔判别法 | 设 $\sum b_n(x)$ 在 $I$ 上一致收敛，在 $I$ 中任意取定一个 $x$，数列 $\{a_n(x)\}$ 关于 $n$ 单调，且 $\{a_n(x)\}$ 在 $I$ 上一致有界，则 $\sum a_n(x)b_n(x)$ 在 $I$ 上一致收敛 |
| 狄利克雷判别法 | 设 $\sum u_n(x)$ 的部分和函数列 $u_n(x) = \sum_{k=1}^{n} u_k(x) (n=1,2,\cdots)$ 在 $I$ 上一致有界；<br />① 对于每一个 $x \in I, \{v_n(x)\}$ 关于 $n$ 单调；<br />② 在 $I$ 上 $v_n(x) \xrightarrow{n \to \infty} 0$。<br />则级数 $\sum u_n(x)v_n(x)$ 在 $I$ 上一致收敛 |



## 一致收敛函数列和函数项级数的性质

| 名称   | 内容 |
| ------ | ---- |
| 定理   | 设函数列 $\{f_n(x)\}$ 在 $(a, x_0) \cup (x_0, b)$ 上一致收敛于 $f(x)$，且对每个 $n$，$\lim_{x \to x_0} f_n(x) = a_n$，则 $\lim_{n \to \infty} a_n$ 和 $\lim_{x \to x_0} f(x)$ 均存在且相等，即 $\lim_{n \to \infty} a_n = \lim_{x \to x_0} \lim_{n \to \infty} f_n(x)$。 |
| 连续性 | 若函数列 $\{f_n(x)\}$ 在区间 $I$ 上一致收敛，且每一项都连续，则其极限函数 $f$ 在 $I$ 上也连续 |
| 可积性 | 若函数列 $\{f_n(x)\}$ 在区间 $[a, b]$ 上一致收敛，且每一项都连续，则 $\int_a^b \lim_{n \to \infty} f_n(x) dx = \lim_{n \to \infty} \int_a^b f_n(x) dx$，即在一致收敛条件下，极限运算与积分运算的顺序可交换 |
| 可微性 | 设 $\{f_n(x)\}$ 为定义在 $[a, b]$ 上的函数列，若 $x_0 \in [a, b]$ 为 $\{f_n(x)\}$ 的收敛点，$f_n(x)$ 的每一项在 $[a, b]$ 上都有连续导数，且 $\{f_n'(x)\}$ 在 $[a, b]$ 上一致收敛，则 $\frac{d}{dx} \left(\lim_{n \to \infty} f_n(x)\right) = \lim_{n \to \infty} \frac{d}{dx} f_n(x)$，即在一致收敛条件下，极限运算与求导运算的顺序可交换 |
| 连续性 | 若函数项级数 $\sum u_n(x)$ 在区间 $[a, b]$ 上一致收敛，且每一项都连续，则其和函数在 $[a, b]$ 上也连续 |
| 逐项积分 | 若函数项级数 $\sum u_n(x)$ 在 $[a, b]$ 上一致收敛，且每一项 $u_n(x)$ 都连续，则 $\int_a^b \sum_{n=1}^{\infty} u_n(x) dx = \sum_{n=1}^{\infty} \int_a^b u_n(x) dx$ |
| 逐项求导 | 若函数项级数 $\sum u_n(x)$ 在 $[a, b]$ 上每一项都有连续的导函数，$x_0 \in [a, b]$ 为 $\sum u_n(x)$ 的收敛点，且 $\sum u_n'(x)$ 在 $[a, b]$ 上一致收敛，则 $(\sum u_n(x))' = \sum u_n'(x), (x \in [a, b])$（$[a, b]$ 换成 $(a, b)$ 也对） |
