# 第十五章 傅里叶级数

## 傅里叶级数

### 1. 傅里叶级数的相关概念
|名称|定义(性质)|
| ---- | ---- |
|三角级数|(1)由三角函数列$\{1,\cos x,\sin x,\cos 2x,\cdots,\cos nx,\sin nx,\cdots\}$所产生的一般形式的三角级数为$\frac{a_{0}}{2}+\sum_{n = 1}^{\infty}(a_{n}\cos nx + b_{n}\sin nx)$。<br />(2)如果$a_{n}=0(n = 1,2,\cdots)$或$b_{n}=0(n = 0,1,2,\cdots)$，则相应的三角级数称为正弦级数(或余弦级数)。<br />(3)若级数$\frac{\abs{a_{0}}}{2}+\sum_{n = 1}^{\infty}(\abs{a_{n}} + \abs{b_{n}})$收敛，则三角级数在整个数轴上绝对收敛且一致收敛|
|正交性|(1)设函数$f(x),g(x)$在$[a,b]$上有定义，若$\int_{a}^{b}f(x)g(x)dx = 0$，则称$f(x),g(x)$在$[a,b]$上是正交的。<br />(2)三角函数列中，任意两个不同函数的乘积在$[-\pi,\pi]$或$[0,2\pi]$上积分为零，则称该三角函数列在$[-\pi,\pi]$或$[0,2\pi]$有正交性|
|傅里叶系数(以$2\pi$为周期)|设函数$f(x)$在区间$[-\pi,\pi]$上可积，则<br />$ a_{n}=\frac{1}{\pi}\int_{-\pi}^{\pi}f(x)\cos nxdx\ (n = 0,1,2,\cdots)$<br />$b_{n}=\frac{1}{\pi}\int_{-\pi}^{\pi}f(x)\sin nxdx\ (n = 1,2,\cdots)$<br />称为$f(x)$的傅里叶系数|
|傅里叶级数(以$2\pi$为周期)|以$a_{0},a_{n},b_{n}(n = 1,2,\cdots)$为系数的三角级数$\frac{a_{0}}{2}+(a_{1}\cos x + b_{1}\sin x)+\cdots+(a_{n}\cos nx + b_{n}\sin nx)+\cdots=\frac{a_{0}}{2}+\sum_{n = 1}^{\infty}(a_{n}\cos nx + b_{n}\sin nx)$称为函数$f(x)$的傅里叶级数，记作$f(x)\sim\frac{a_{0}}{2}+\sum_{n = 1}^{\infty}(a_{n}\cos nx + b_{n}\sin nx)$|

**注**：若级数$\frac{|a_{0}|}{2}+\sum_{n = 1}^{\infty}(|a_{n}| + |b_{n}|)$收敛，因为$|a_{n}\cos nx + b_{n}\sin nx|\leq|a_{n}|+|b_{n}|$，所以$\frac{a_{0}}{2}+\sum_{n = 1}^{\infty}(a_{n}\cos nx + b_{n}\sin nx)$在数轴上绝对收敛且一致收敛。

### 2. 傅里叶级数收敛定理
(1)定理：设以$2\pi$为周期的函数$f(x)$在区间$[-\pi,\pi]$上按段光滑，则在每一点$x\in[-\pi,\pi]$，$f(x)$的傅里叶级数收敛于$f(x)$在点$x$的左、右极限的算术平均值，即
$f(x + 0)+f(x - 0)=\frac{a_{0}}{2}+\sum_{n = 1}^{\infty}(a_{n}\cos nx + b_{n}\sin nx)$，
其中$a_{n}$和$b_{n}$为$f(x)$的傅里叶系数。",
（2）推论：若 $f(x)$ 是以 $2\pi$ 为周期的连续函数，且在 $[-\pi,\pi]$ 上按段光滑，则 $f(x)$ 的傅里叶级数在 $(-\infty,+\infty)$ 上收敛于 $f(x)$

小提示：设$f(x)$是周期为$2\pi$的周期函数，如果它满足：在一个周期内连续或只有有限个第一类间断点，在一个周期内至多只有有限个极值点，则$f(x)$的傅里叶级数收敛，并且有：
1. 当$x$是$f(x)$的连续点时，级数收敛于$f(x)$。
2. 当$x$是$f(x)$的间断点时，级数收敛于$\frac{1}{2}[f(x-0)+f(x+0)]$。


## 以 $2l$ 为周期的函数的展开式

| 名称 | 内容 |
| --- | --- |
| 傅里叶级数 | 设函数 $f(x)$ 及 $f'(x)$ 在区间 $[-l, l]$ 上逐段连接，则在连续点上，$f(x)$ 有傅里叶级数 $f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos \frac{n\pi x}{l} + b_n \sin \frac{n\pi x}{l} \right)$ 其中 $a_n, b_n$ 为 $f(x)$ 的傅里叶系数 |
| 以 $2l$ 为周期的函数的傅里叶级数 | $a_n = \frac{1}{l} \int_{-l}^{l} f(x) \cos \frac{n\pi x}{l} dx, n = 0, 1, 2, \ldots$ $b_n = \frac{1}{l} \int_{-l}^{l} f(x) \sin \frac{n\pi x}{l} dx, n = 1, 2, \ldots$ 若函数 $f(x)$ 在 $[-l, l]$ 上按段光滑，则同样可由收敛定理知道 $f(x+0) + f(x-0) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos \frac{n\pi x}{l} + b_n \sin \frac{n\pi x}{l} \right)$ |
| 偶函数 | 设 $f(x)$ 是以 $2l$ 为周期的偶函数，或是定义在 $[-l, l]$ 上的偶函数，则在 $[-l, l]$ 上，$f(x) \cos nx$ 是偶函数，$f(x) \sin nx$ 是奇函数。因此，$f(x)$ 的傅里叶系数 $a_n = \frac{2}{l} \int_{0}^{l} f(x) \cos \frac{n\pi x}{l} dx, b_n \equiv 0$ 且连续点处有 $f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} a_n \cos \frac{n\pi x}{l}$（也叫余弦级数） |
| 奇函数 | 设 $f(x)$ 是以 $2l$ 为周期的奇函数，或是定义在 $[-l, l]$ 上的奇函数，则有 $a_n = 0 (n = 0, 1, 2, \ldots)$， $b_n = \frac{2}{\pi} \int_{0}^{\pi} f(x) \sin nx dx (n = 1, 2, \ldots)$ 且连续点处有 $f(x) = \sum_{n=1}^{\infty} b_n \sin \frac{n\pi x}{l}$（也叫正弦级数） |


小提示：求傅里叶级数的步骤：①确定函数的奇偶性、连续点与不连续点，必要时画出至少一个周期的函数图像；②根据周期、奇偶，写出相应的傅里叶系数；③写出相应的傅里叶级数；④验证收敛条件，写出傅里叶展开式.注意间断点处的收敛状态。

## 收敛定理的证明

1. 贝塞尔不等式

若函数 $f(x)$ 在 $[-\pi, \pi]$ 上可积, 则

$\frac{a_0^2}{2} + \sum_{n=1}^{\infty} (a_n^2 + b_n^2) \leq \frac{1}{\pi} \int_{-\pi}^{\pi} f^2(x) dx,$ 其中 $a_n, b_n$ 为 $f(x)$ 的傅里叶级数。

2. 黎曼—勒贝格定理

若 $f(x)$ 为 $[-\pi, \pi]$ 上的可积函数, 则

$$
\lim_{n \to \infty} \int_{-\pi}^{\pi} f(x) \cos nx \, dx = 0, \lim_{n \to \infty} \int_{-\pi}^{\pi} f(x) \sin nx \, dx = 0.
$$
